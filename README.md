## Installation conditions 编译安装先决条件  
- G++  
- QT  
## How install  
Please enter in the terminal:(If you are using Windows, change xxxx.o to xxx.exe)   
```Shell  
g++ init.cpp -o init.o  
g++ kernel.cpp -o init.o   
```  
Enjoy your game!
